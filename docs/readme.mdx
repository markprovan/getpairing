# Get Pairing

### The complete guide to Pair Programming

Get Pairing is a resource for engineers, designers, product owners and everyone involved in the software delivery process. You'll find guides on the benefits, tips for getting started and even advice for those experienced with pairing. Not sure where to start? [Try What is Pairing?](/what-is-pairing).
